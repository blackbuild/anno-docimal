# Keep canonical documentation with the source

`README.md` is AnnoDocimal's concise landing page and quick start, version-controlled files under `docs/` own detailed
user and architecture documentation, and generated Javadocs provide API reference. A separate mutable wiki is not a
canonical source. This aligns with the intended direction away from wiki-owned KlumAST documentation and supports a
future common scaffolding model across AnnoDocimal, KlumAST, klum-cast, and additional Klum-family projects. Selecting or
implementing that shared scaffolding remains a separate cross-repository decision.
