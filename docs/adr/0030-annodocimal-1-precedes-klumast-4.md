# Release AnnoDocimal 1.0 before KlumAST 4.0

AnnoDocimal 1.0 is a hard release prerequisite for KlumAST 4.0. KlumAST development may consume AnnoDocimal pre-release
versions, but KlumAST 4.0 must not stabilize or release its documentation integration against provisional AnnoDocimal
0.x APIs. Cross-repository implementation sequencing and any KlumAST issue or release-plan changes must be handled in
the owning repository; this decision does not merge AnnoDocimal's product boundary with KlumAST's.
