# Target structured protocol evolution for 2.0

Issue #30 does not block AnnoDocimal 1.0. Version 1.0 stabilizes the current carrier's supported behavior and the
additive-evolution promise. The structured annotation initiative targets a future 2.0 release because it is a major
ecosystem change even if a compatibility bridge avoids a strict source or binary break. Its mandatory migration plan
must cover supporting projects, including the AnnoDoc IntelliJ plugin, as well as mixed protocol versions. This session
does not design that plan.
